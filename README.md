# DM2350 - Human Perception for Information Technology

## Web application (React)
This is a change detection test developed during the course DM2350 Human Perception for Information Technology @ KTH 2018. The application was made to investigate how color affects visual working memory in an UI interface. 

### Technologies
* React
* Bootstrap
* Google Forms (Store and analyze test data)

### Installing
```
npm install
npm start
```
